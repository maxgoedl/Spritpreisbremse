# Spritpreisbremse

Interaktive Vite/React-App zu Benzin- und Dieselpreisen in Österreich und Deutschland rund um die österreichische Spritpreisbremse am 1. April 2026.

## Lokal starten

```bash
npm install
npm run dev
```

## GitHub Pages

Die App ist für das Repository `maxgoedl/Spritpreisbremse` vorbereitet und nutzt daher in `vite.config.js` den Base-Pfad `/Spritpreisbremse/`.

Damit GitHub Pages funktioniert:

1. In GitHub: `Settings` -> `Pages` -> `Source` = `GitHub Actions`
2. Push auf `main`
3. Danach ist die App unter `https://maxgoedl.github.io/Spritpreisbremse/` erreichbar.
